# Grid TUI Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Replace the accordion TUI with a spreadsheet-like grid that matches the HTML mockup aesthetic exactly.

**Architecture:** New Textual app (`tapes/ui/grid.py`) renders a flat list of rows using Rich Text objects inside a single scrollable widget. Each row is a line of fixed-width column spans. Cursor, selection, and highlights are applied by re-rendering affected rows. State lives in a `GridState` dataclass that the widget reads from.

**Tech Stack:** Textual 3.x (already a dependency), Rich Text for styled row rendering, Textual CSS for layout chrome (footer, separators). No new dependencies.

**Visual reference:** `docs/mockups/002-mockup.html`

## Milestones

This plan is broken into sequential milestones. Each milestone produces a working, testable increment. **Stop after each milestone for review.**

- **M1: Grid rendering + navigation** -- render rows with correct columns and colors, crosshair cursor, arrow key navigation
- **M2: Selection** -- column-locked vertical selection with `v`, mossy green highlight
- **M3: Inline edit + manual status** -- `e` to edit a field, `!!` badge, purple for edited fields
- **M4: Query mock + auto-accept** -- `q` triggers mock TMDB results, confident matches auto-replace with `**` and cyan
- **M5: Uncertain matches** -- `??` sub-rows, accept/reject with enter/backspace
- **M6: Process mode** -- `p` swaps to destination paths, confirmation screen
- **M7: Shift+E modal** -- all-fields editor overlay

Each milestone is a commit point. We build M1 first, review, then proceed.

## Shared conventions

- All new UI code goes in `tapes/ui/`. The old `app.py`, `split_modal.py`, `merge_modal.py`, `file_editor.py` stay until we're ready to delete them.
- Tests go in `tests/test_ui/`. Use Textual's `app.run_test()` async pattern.
- The grid works on `list[GridRow]` -- a flat view model, not `ImportGroup` directly. A builder function converts `list[ImportGroup]` to `list[GridRow]`.
- Column widths (in `ch` units from the mockup): status=5, filepath=52, title=16, year=6, season=4, episode=4, episode_title=32.
- Colors use Rich style strings matching the mockup: `#ddd` (white/primary), `#888` (text), `#555` (dim), `#333` (very dim), `#5a9` (green), `#ca3` (yellow), `#c55` (red), `#6bc` (cyan), `#a78bfa` (purple).

```python
"""View models for the grid TUI."""
from __future__ import annotations

from dataclasses import dataclass, field
from enum import Enum, auto
from pathlib import Path

from tapes.models import FileEntry, FileMetadata, ImportGroup


class RowKind(Enum):
    FILE = auto()
    BLANK = auto()
    MATCH = auto()  # M5: uncertain match sub-row


class RowStatus(Enum):
    RAW = ".."       # guessit only
    AUTO = "**"      # auto-accepted TMDB
    UNCERTAIN = "??" # uncertain match
    EDITED = "!!"    # user-edited


@dataclass
class GridRow:
    """One visual row in the grid. May be a file or a blank separator."""
    kind: RowKind
    entry: FileEntry | None = None
    group: ImportGroup | None = None
    status: RowStatus = RowStatus.RAW
    edited_fields: set[str] = field(default_factory=set)

    @property
    def is_video(self) -> bool:
        return self.entry is not None and self.entry.role == "video"

    @property
    def is_companion(self) -> bool:
        return self.entry is not None and self.entry.role != "video"

    def _meta(self) -> FileMetadata:
        if self.entry and self.entry.metadata:
            return self.entry.metadata
        if self.group:
            return self.group.metadata
        return FileMetadata()

    @property
    def title(self) -> str | None:
        return self._meta().title

    @property
    def year(self) -> int | None:
        return self._meta().year

    @property
    def season(self) -> int | None:
        return self._meta().season

    @property
    def episode(self) -> int | list[int] | None:
        return self._meta().episode

    @property
    def episode_title(self) -> str | None:
        return None  # TMDB only, not from guessit

    @property
    def filepath(self) -> str:
        if self.entry:
            return str(self.entry.path)
        return ""


def build_grid_rows(groups: list[ImportGroup]) -> list[GridRow]:
    """Convert ImportGroups into a flat list of GridRows with blank separators."""
    rows: list[GridRow] = []
    for i, group in enumerate(groups):
        if i > 0:
            rows.append(GridRow(kind=RowKind.BLANK))
        for entry in group.files:
            rows.append(GridRow(kind=RowKind.FILE, entry=entry, group=group))
    return rows
```

```python
"""Render a GridRow as a Rich Text line with fixed-width columns."""
from __future__ import annotations

from pathlib import PurePosixPath
from rich.text import Text

from tapes.ui.models import GridRow, RowKind, RowStatus

# Column widths in characters (matching mockup ch units)
COL_WIDTHS: dict[str, int] = {
    "status": 5,
    "filepath": 52,
    "title": 16,
    "year": 6,
    "season": 4,
    "episode": 4,
    "episode_title": 32,
}

# Column keys in display order (for cursor indexing)
FIELD_COLS: list[str] = ["title", "year", "season", "episode", "episode_title"]

# Status badge styles
_BADGE_STYLES: dict[RowStatus, tuple[str, str]] = {
    RowStatus.RAW: ("..", "#555"),
    RowStatus.AUTO: ("**", "#5a9"),
    RowStatus.UNCERTAIN: ("??", "#ca3"),
    RowStatus.EDITED: ("!!", "#a78bfa"),
}

# Highlight backgrounds
BG_ROW_CUR = "#1e1e1e"
BG_COL_HI = "#1e1e1e"
BG_CELL_CUR = "#2c2c2c"
BG_CELL_SEL = "#363636"
BG_ROW_SEL = "#1c1c1c"
BG_CELL_SEL_CUR = "#3c3c3c"


def _pad(text: str, width: int) -> str:
    """Pad or truncate text to exactly width characters."""
    if len(text) >= width:
        return text[: width - 1] + "\u2026"
    return text + " " * (width - len(text))


def _col(t: Text, value: str, width: int, style: str, bg: str | None = None) -> None:
    """Append a fixed-width column span to a Text object."""
    padded = _pad(value, width)
    full_style = style
    if bg:
        full_style = f"{style} on {bg}"
    t.append(padded, style=full_style)


def render_row(
    row: GridRow,
    cursor_col: int | None,
    is_cursor_row: bool,
    *,
    selected_cols: set[int] | None = None,
    is_sel_cursor_row: bool = False,
) -> Text:
    """Render one grid row as a Rich Text object.

    Args:
        row: The GridRow to render.
        cursor_col: Index into FIELD_COLS for the cursor column (None = no column highlight).
        is_cursor_row: Whether this is the row the cursor is on.
        selected_cols: Set of column indices that are selected (for selection mode).
        is_sel_cursor_row: Whether this is the cursor row during selection.
    """
    if selected_cols is None:
        selected_cols = set()

    t = Text()

    if row.kind == RowKind.BLANK:
        # Render blank row with column slots for highlights
        _col(t, "", COL_WIDTHS["status"], "#333", bg=None)
        _col(t, "", COL_WIDTHS["filepath"], "#333", bg=None)
        for i, col_name in enumerate(FIELD_COLS):
            bg = None
            if i == cursor_col:
                bg = BG_COL_HI
            if i in selected_cols:
                bg = BG_CELL_SEL
            _col(t, "", COL_WIDTHS[col_name], "#333", bg=bg)
        return t

    # Determine base styling
    is_comp = row.is_companion
    base_style = "#555" if is_comp else "#888"
    bright_style = "#888" if is_comp else "#ddd"

    # Row background
    row_bg = None
    if is_cursor_row:
        row_bg = BG_ROW_CUR
    elif is_sel_cursor_row:
        row_bg = BG_ROW_SEL
    elif any(i in selected_cols for i in range(len(FIELD_COLS))):
        row_bg = BG_ROW_SEL

    # Status badge
    badge_text, badge_style = _BADGE_STYLES.get(row.status, ("..", "#555"))
    _col(t, badge_text.rjust(COL_WIDTHS["status"] - 1) + " ", COL_WIDTHS["status"],
         badge_style, bg=row_bg)

    # Filepath
    fp = row.filepath
    p = PurePosixPath(fp)
    if len(p.parts) > 1 and row.is_video:
        dir_part = str(p.parent) + "/"
        name_part = p.name
        padded_fp = _pad(dir_part + name_part, COL_WIDTHS["filepath"])
        fp_text = Text()
        fp_text.append(dir_part, style=f"#555 on {row_bg}" if row_bg else "#555")
        remaining = padded_fp[len(dir_part):]
        fp_text.append(remaining, style=f"{bright_style} on {row_bg}" if row_bg else bright_style)
        t.append_text(fp_text)
    else:
        style = base_style if is_comp else bright_style
        _col(t, _pad(fp, COL_WIDTHS["filepath"]), COL_WIDTHS["filepath"],
             style, bg=row_bg)

    # Metadata columns
    values = [
        row.title or "",
        str(row.year) if row.year else "",
        str(row.season) if row.season is not None else "",
        str(row.episode) if row.episode is not None else "",
        row.episode_title or "",
    ]

    for i, (col_name, value) in enumerate(zip(FIELD_COLS, values)):
        # Determine style
        if row.status == RowStatus.EDITED and col_name in row.edited_fields:
            style = "#a78bfa"  # purple for edited
        elif row.status == RowStatus.AUTO:
            style = "#6bc" if value else base_style  # cyan for TMDB data
        else:
            style = base_style if is_comp else bright_style if value else base_style

        # Determine background
        bg = row_bg
        if is_cursor_row and i == cursor_col:
            bg = BG_CELL_CUR
            style = "#fff"
        elif i == cursor_col:
            bg = BG_COL_HI
        if i in selected_cols:
            if is_sel_cursor_row and i == cursor_col:
                bg = BG_CELL_SEL_CUR
                style = "#fff"
            else:
                bg = BG_CELL_SEL

        _col(t, value, COL_WIDTHS[col_name], style, bg=bg)

    return t
```

```python
"""Grid TUI app for tapes."""
from __future__ import annotations

from textual.app import App, ComposeResult
from textual.binding import Binding
from textual.widgets import Static, Footer
from textual.containers import VerticalScroll
from rich.text import Text

from tapes.models import ImportGroup
from tapes.ui.models import GridRow, RowKind, build_grid_rows
from tapes.ui.render import render_row, FIELD_COLS


class GridWidget(Static):
    """Renders the entire grid as styled text lines."""

    def __init__(self, rows: list[GridRow], **kwargs) -> None:
        super().__init__(**kwargs)
        self.rows = rows
        self._cursor_row = 0
        self._cursor_col = 0

    def render_grid(self) -> Text:
        """Re-render all rows into a single Text block."""
        out = Text()
        for i, row in enumerate(self.rows):
            if i > 0:
                out.append("\n")
            line = render_row(
                row,
                cursor_col=self._cursor_col,
                is_cursor_row=(i == self._cursor_row),
            )
            out.append_text(line)
        return out

    def refresh_grid(self) -> None:
        self.update(self.render_grid())


class GridApp(App):
    """Spreadsheet-like grid TUI for reviewing imports."""

    TITLE = "tapes"
    CSS = """
    Screen { background: #111111; }
    #grid-scroll { background: #111111; }
    #grid { padding: 1 2; }
    #header-line { padding: 0 2; color: #555555; height: 1; }
    Footer { background: #0e0e0e; color: #444444; }
    """

    BINDINGS = [
        Binding("up", "cursor_up", "Up", show=False),
        Binding("down", "cursor_down", "Down", show=False),
        Binding("left", "cursor_left", "Left", show=False),
        Binding("right", "cursor_right", "Right", show=False),
        Binding("q", "quit", "Quit", show=True),
    ]

    def __init__(self, groups: list[ImportGroup], **kwargs) -> None:
        super().__init__(**kwargs)
        self._groups = groups
        self._rows = build_grid_rows(groups)
        self._grid: GridWidget | None = None

    @property
    def cursor_row(self) -> int:
        return self._grid._cursor_row if self._grid else 0

    @property
    def cursor_col(self) -> int:
        return self._grid._cursor_col if self._grid else 0

    def compose(self) -> ComposeResult:
        yield Static(self._render_header(), id="header-line")
        with VerticalScroll(id="grid-scroll"):
            self._grid = GridWidget(self._rows, id="grid")
            yield self._grid
        yield Footer()

    def on_mount(self) -> None:
        # Skip to first file row if row 0 is blank
        self._skip_to_file(1)
        self._grid.refresh_grid()

    def _render_header(self) -> Text:
        t = Text()
        t.append("tapes import", style="#555")
        t.append(" ")
        t.append("./downloads", style="#ddd")
        return t

    def _skip_to_file(self, direction: int) -> None:
        """Ensure cursor is on a FILE row, skipping blanks in given direction."""
        if not self._grid:
            return
        row = self._grid._cursor_row
        while 0 <= row < len(self._rows) and self._rows[row].kind != RowKind.FILE:
            row += direction
        if 0 <= row < len(self._rows):
            self._grid._cursor_row = row

    def _file_rows(self) -> list[int]:
        """Return indices of all FILE rows."""
        return [i for i, r in enumerate(self._rows) if r.kind == RowKind.FILE]

    def action_cursor_down(self) -> None:
        if not self._grid:
            return
        file_rows = self._file_rows()
        cur = self._grid._cursor_row
        # Find next file row after current
        for idx in file_rows:
            if idx > cur:
                self._grid._cursor_row = idx
                self._grid.refresh_grid()
                return

    def action_cursor_up(self) -> None:
        if not self._grid:
            return
        file_rows = self._file_rows()
        cur = self._grid._cursor_row
        # Find previous file row before current
        for idx in reversed(file_rows):
            if idx < cur:
                self._grid._cursor_row = idx
                self._grid.refresh_grid()
                return

    def action_cursor_right(self) -> None:
        if not self._grid:
            return
        if self._grid._cursor_col < len(FIELD_COLS) - 1:
            self._grid._cursor_col += 1
            self._grid.refresh_grid()

    def action_cursor_left(self) -> None:
        if not self._grid:
            return
        if self._grid._cursor_col > 0:
            self._grid._cursor_col -= 1
            self._grid.refresh_grid()
```

### M2 revision notes

**Selection model: selecting mode with pause/resume.** First attempt was
explicit per-cell toggle, but it was tedious. Changed to a mode where `v`
starts selecting, arrows add cells, `v` pauses, `v` resumes. Non-adjacent
selection via pause/navigate/resume or `V` for groups.

**Monochrome gray replaced with mossy green.** Gray was indistinguishable
from cursor/column highlights. Green hue makes selection immediately visible.

**`_SEL_TEXT` brightness mapping.** Dim text (`#555555`) was unreadable on
green backgrounds. Lookup table maps each text color to a readable variant.

**Cursor never changes text color.** Only background changes on hover.
